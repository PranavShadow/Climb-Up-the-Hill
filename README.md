# Pranav Up the Hill

## 🎮 About  
**Pranav Up the Hill** is a basic 2D Python game where the goal is simple: avoid obstacles and reach the red flag to win.

## 🚀 How to Play  
- Use the arrow keys to move.  
- Dodge the obstacles.  
- Reach the red flag to complete the level.

## 🛠️ Run the Game  
Make sure Python is installed, then:

```bash
python pranav_up_the_hill.py
```
